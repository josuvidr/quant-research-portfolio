# Systematic Equity Factor Strategy

**Objective:**  
Construct and evaluate a simple long/short equity factor strategy (e.g., momentum or value) using historical data.

**Key components:**  
- Signal construction  
- Portfolio formation  
- Backtest engine (returns, turnover, costs)  
- Sharpe, Sortino, drawdown, hit-rate  
- Risk diagnostics (exposures, beta, factor correlations)  
- Sensitivity tests
